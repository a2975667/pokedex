# Pokédex Stage 007: Basic Search Bar

This stage starts the frontend build. The default homepage is replaced with a custom Pokédex page that includes a search form, but it does not render results yet.

## Learning Goals

- Convert the homepage into a client component
- Manage local form state with `useState`
- Build the shell of an application before wiring in data

## What Exists In This Stage

- All backend routes from stage `006`
- A custom homepage in `app/page.tsx`
- A search input and button
- No result grid yet

## Running This Stage

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Suggested Teaching Focus

This branch marks the transition from backend-first work to frontend-first work. Students can focus on UI structure before mixing in fetching logic.

## Next Stage

Stage `008` connects the page to the API and renders Pokémon cards in a grid.
